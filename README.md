# ⚙️ Functional Treat - Data Processing Toolkit

## 📖 Description
Functional Treat is a Python-based menu-driven program that demonstrates the use of functions, recursion, lambda expressions, and built-in functions. It allows users to input data and perform various operations like sorting, filtering, and statistical analysis.

---

## 🚀 Features

### 📥 Input Data
- Enter a list of numbers (1D array)
- Data stored using list and `map()` function

### 📊 Data Summary
- Total elements
- Minimum value
- Maximum value
- Sum of values
- Average value

### 🔁 Factorial (Recursion)
- Calculates factorial using recursive function

### 🔍 Filter Data
- Filters values based on threshold using `lambda` and `filter()`

### 🔃 Sort Data
- Ascending order
- Descending order

### 📈 Dataset Statistics
- Returns multiple values:
  - Min
  - Max
  - Sum
  - Average

### 🔁 Menu Driven System
- Continuous loop until user exits

---

## 🧑‍💻 Technologies Used
- Python
  - Functions
  - Recursion
  - Lambda Functions
  - Built-in Functions (`len()`, `min()`, `max()`, `sum()`)
  - List Operations
